# AZ-900 Practice Simulator

Simulador interativo para estudo da certificação **Microsoft Azure Fundamentals (AZ-900)**.

## 📚 Sobre o projeto

Este projeto foi desenvolvido para praticar conceitos de computação em nuvem e ajudar no preparo para o exame AZ-900.

O simulador apresenta perguntas de múltipla escolha e calcula automaticamente a pontuação final do usuário.

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- HTML
- CSS

## 🧠 Funcionalidades

- Simulado com perguntas sobre Azure
- Seleção de respostas
- Feedback de resposta correta/incorreta
- Barra de progresso
- Cálculo de pontuação final
- Interface simples e interativa

## ▶️ Como executar o projeto

Clone o repositório:

git clone https://github.com/matheus619silva/az900-simulator.git

Entre na pasta:

cd az900-simulator

Instale dependências:

npm install

Execute o projeto:

npm run dev
